

Projet minimal Spring MVC.
==========================
Ce projet a été créé comme décrit dans http://www.programcreek.com/2014/02/spring-mvc-helloworld-using-maven-in-eclipse/.

Basically:
  1) utiliser le archetype org.apache.maven.archetype  .... maven-archetype-webapp
  2) Ajouter les jar Spring dans pom.xml
  3) Modifier web.xml pour ajouter le DispatcherSErvlet de Spring, et le dispatcher-servlet.xml
  4) Ajouter un Controller.java
  5) Ajouter un index.jsp et un WEB-INFviews/Helloworld.jsp
  6) Invoker localhost:8080/HelloSpringMvc (le nom HelloSpringMvc vient du pom.xml)


(je voulais aussi JPA, mais viendra apres).
